# 股票小助手
一款基于flutter，可以在系统托盘显示股票信息的macOS桌面应用，实乃小韭菜打工必备。
# 使用说明
1. 添加股票并在设置中开启托盘显示，默认在系统托盘显示股票列表前三的价格

2. 在股票列表<font color=red>长按拖动</font>调整顺序
